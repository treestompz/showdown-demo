Showdown is a platform that enables CS:GO players to 1v1 each other in-game for with skins on the line.

This is a demo of the many features of the platform.





Use or buttons below to navigate.

How it works

Find Opponent

Players find an opponent to challenge to a showdown (a 1v1 match) with our advanced Matchmaking system.

Customize

Choose the aim map, weapon, rounds, round timer, and more.

Stake coins

Players stake coins (currency) on the showdown. Winner takes all.

Frag out

Players play the match on one of our NA or EU 128 tick servers that are always available. The system is designed so players don’t have to wait for server startup time.

Matchmaking / Showdowns Page

This isn’t the home page, but it’s the most used page on the site as this is where players find their opponents. On this page players can post a showdown that has the map, weapon, etc. The video below demonstrates a player posting a showdown using our awesome custom form and then another user accepting the challenge.

Playing the showdown

Showdowns are played on one of our 128 tick servers with locations in NA and EU. However, what makes our servers special is that the game plugin running on them is engineered to be scalable and highly available. Instead of dedicating entire source servers to a single showdown, we can have as many showdowns running on one source server instance as we want. Not only does this drastically reduce overhead, it means that, as shown in the demo video below, showdown servers are ready instantly. There is no need to wait for an instance to spin up.

Post showdown (Reporting)

After a showdown is completed, the loser has 10 minutes to file a report if they think their opponent was cheating. If those 10 minutes pass and no report is filed, the showdown is marked as finished and the winner is awarded their coins. All showdowns are recorded as demos in 128 tick. If a report is filed it goes into the reports section of the admin panel. The demo is also attached to the report for review.

Admin Report View

This is the admin view of a user's report. It includes controls to ban users, change the winner, cancel the match, download a full replay of the match, and more.

Spending your coins (Item Store)

Like many sites that deal with CS:GO skins, Showdown has a robust system for handling CS:GO skins. Players can deposit skins for coins, withdraw skins for coins, and bet on showdowns with coins. Our bots are battle-tested and account for all the randomness Steam can throw at them.

More pages

Let's take a look at some other pages on the site.

Home Page (Dashboard)

This is the home page when users are signed in. It acts as a dashboard of sort. Users can see pending/past reports, pending/past showdowns, statistics, news, and more.

Reports Page

On this page the user gets a more detailed look at their pending report. A report can lead to the winner changing, coins being returned, or a user being banned. The admin view shows additional details and automatically links the demo file

Leaderboards Page

This page contains a simple leaderboard to incentivize players to play more showdowns to increase their ranking.

Private Match / Friends Page

This page allows user to create private showdowns with their Steam friends.
